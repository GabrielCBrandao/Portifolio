💼 Portfólio Pessoal - Gabriel Brandão
🚀 Sobre o Projeto

Este é o meu portfólio pessoal desenvolvido com HTML5, CSS3 e JavaScript, criado com o objetivo de apresentar minha trajetória como Desenvolvedor Front-End, minhas habilidades técnicas e alguns dos projetos que desenvolvi ao longo da minha jornada na programação.

O projeto foi construído com foco em uma experiência moderna e agradável para o usuário, utilizando animações, efeitos visuais e uma navegação intuitiva.

🎯 Objetivos
Apresentar minha experiência profissional e técnica.
Exibir projetos desenvolvidos e publicados.
Facilitar o contato com recrutadores e clientes.
Demonstrar conhecimentos em HTML, CSS e JavaScript.
✨ Funcionalidades
✅ Menu de navegação fixo
